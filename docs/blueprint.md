# **App Name**: LexiCapture

## Core Features:

- Word Capture: Capture English words and their part of speech using the phone's camera.
- Definition Lookup: Use a large language model to get the Chinese definition of the captured word and part of speech.
- Data Storage: Store the captured word, part of speech, Chinese definition, and the date/time of capture in a database.
- Review: Enable reviewing words, definitions, and usage examples.

## Style Guidelines:

- Use a clean and modern color palette with a focus on readability.
- Employ a clear and legible font for displaying words and definitions.
- Organize information in a visually appealing and intuitive manner.
- Use intuitive icons to represent different functions and features.
